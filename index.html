Trivia Guessing Game

Description

  • This game is a simple guessing game with random trivia facts!

How to Play

  • Each question is multiple choice with 3 possible answers of a, b, or c.
  • Fill in your answer using only lower case letters.
  • Get as many questions right as you can to get your name on the high score board!
